Welcome to the Enterprise AI Risk & Performance Intelligence System!

This is a production-ready full-stack AI system for workforce analytics.

🚀 QUICK START (Choose one):

Option 1: Interactive Setup (Recommended)
    python quickstart.py

Option 2: Manual Start
    1. pip install -r requirements.txt
    2. copy .env.example .env (edit with your credentials)
    3. python api/main.py (in Terminal 1)
    4. streamlit run dashboard/app.py (in Terminal 2)

Option 3: Docker
    docker-compose up
